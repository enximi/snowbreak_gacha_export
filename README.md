# Snowbreak Gacha Export

中文 | [English](doc/README_EN)

尘白禁区抽卡记录导出工具

支持 16:9 分辨率窗口化/无边框

使用OCR识别

只在使用简体中文的国际服测试过

## 功能

- [x] 抽卡记录导出
- [x] 新增抽卡记录合并
- [x] 多账号支持
- [x] 以可读性较好的方式展示记录（records.xlsx）
- [ ] 使用命令行传递运行参数

## 使用方法

下载，运行。

抽卡记录保存在`records/records.json`文件中。

另有`records.xlsx`文件，以可读性较好的方式展示记录。
